| 问题现象 | 主要原因 | 解决办法 |
| :--- | :--- | :--- |
| 叶片快速发黄、萎蔫 | 温度过高、湿度不足，失水过快 | 调整舱室温度至 0℃~4℃，增加环境湿度，用湿毛巾轻盖保鲜袋外保湿 |
| 根部发黏、腐烂 | 预处理时带水、存放积水，通风不良 | 立即分拣丢弃腐烂部分，剩余韭菜更换吸水纸，重新分装透气存放 |
| 出现异味、串味 | 与洋葱/大蒜等同放，或密封不透气 | 立即分拣，用清水快速冲洗后优先加工食用；后续存放时做好隔离，使用带透气孔的容器 |
| 叶片发黑、软烂 | 被重物挤压、机械损伤，或存放过久 | 挑除损伤软烂的部分，剩余部分优先加工使用；后续存放时竖直摆放，避免挤压堆叠 |
